📖 이미지 최적화 가이드 01
블로그 이미지 최적화 완벽 가이드 — SEO 점수를 올리는 이미지 세팅 A to Z
이미지 하나를 잘못 올리면 블로그 전체 속도가 느려집니다. 포맷 선택부터 파일명, 용량까지 구글이 좋아하는 이미지 세팅법을 한 번에 정리했습니다.
2025년 7월
이미지가 SEO에 영향을 주는 이유
구글은 2021년부터 Core Web Vitals를 검색 순위 신호로 공식 채택했습니다. 그 중 LCP(Largest Contentful Paint, 최대 콘텐츠 표시 시간)는 페이지에서 가장 큰 이미지가 화면에 나타나는 데 걸리는 시간입니다. 블로그 특성상 이 'LCP 요소'는 대부분 대표 이미지(썸네일)이기 때문에, 이미지 최적화는 곧 SEO 점수와 직결됩니다.
  • LCP 2.5초 이하 → 구글 기준 '좋음' 등급
  • LCP 4.0초 초과 → '나쁨' 등급, 검색 순위 하락 위험
  • 이미지 용량을 절반으로 줄이면 LCP는 평균 0.5~1.5초 단축됩니다
1단계 — 올바른 포맷 선택하기
블로그 이미지에는 WebP 포맷을 강력히 추천합니다. WebP는 구글이 직접 개발한 차세대 이미지 형식으로, 같은 품질의 JPG 대비 25~35%, PNG 대비 최대 50% 더 작은 파일 크기를 제공합니다.

단, 용도별로 포맷을 구분하는 것이 중요합니다.
  • 사진·음식·풍경 이미지 → WebP (가장 작고 빠름)
  • 로고·아이콘·투명 배경 → PNG (투명도 유지 필수)
  • 구형 플랫폼 호환 필요 시 → JPG (호환성 최우선)
💡 체크포인트
티스토리, 네이버 블로그, 워드프레스 모두 WebP를 지원합니다. 2025년 기준 국내 주요 브라우저의 WebP 지원율은 97% 이상입니다. 안심하고 WebP를 사용하세요.
2단계 — 이미지 크기와 용량 줄이기
블로그 본문에 사용하는 이미지는 가로 너비 1200px, 파일 크기 200KB 이하가 이상적입니다. 스마트폰으로 찍은 사진은 보통 4000px 이상, 5~10MB에 달하는데 이를 그대로 올리면 페이지 로딩이 크게 느려집니다.

압축짤은 최대 너비 1920px로 자동 리사이즈하고, WebP 품질 82%로 압축합니다. 대부분의 블로그 이미지에서 육안으로 화질 차이 없이 50~70% 용량을 줄일 수 있습니다.
3단계 — SEO 파일명 설정하기
구글 이미지 검색봇은 파일명을 직접 크롤링합니다. DSC_00123.jpg처럼 의미 없는 파일명보다 대구-동성로-맛집-파스타.webp처럼 키워드가 담긴 파일명이 이미지 검색 노출에 유리합니다.

압축짤의 '파일명 키워드' 기능을 활용하면 업로드한 이미지 전체에 순번을 붙여 일괄 변경할 수 있어 시간을 크게 절약할 수 있습니다.
  • 키워드는 글의 핵심 주제어 사용 (예: 홍대카페, 제주여행, 육아용품)
  • 영어보다 한국어 키워드가 한국 검색에 더 유리
  • 단어 사이 구분은 하이픈(-) 권장
4단계 — 이미지 alt 태그 작성하기
이미지 압축과 파일명 최적화가 끝났다면, 블로그에 업로드한 후 반드시 alt(대체 텍스트)를 작성하세요. 구글은 이미지 자체를 완벽하게 이해하지 못하기 때문에 alt 태그를 통해 이미지 내용을 파악합니다.

좋은 alt 태그 예시:
  • 나쁜 예: 이미지, 사진1
  • 좋은 예: 대구 동성로 이탈리안 레스토랑 파스타 메뉴
