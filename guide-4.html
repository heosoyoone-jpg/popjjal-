2025년 7월
구글은 파일명을 어떻게 활용하나요?
구글 이미지 검색봇(Googlebot Images)은 이미지 파일 자체를 분석하는 것 외에도 파일명, URL, alt 텍스트, 주변 텍스트를 함께 읽어 이미지가 무엇을 나타내는지 파악합니다. 구글 공식 가이드에도 "파일명은 이미지 콘텐츠를 설명하는 데 도움이 됩니다"라고 명시되어 있습니다.

즉, 파일명에 의미 있는 키워드를 넣으면 이미지 검색 결과에서 더 잘 노출될 가능성이 높아집니다. 반대로 DSC_00123.jpg처럼 카메라가 자동으로 붙인 파일명은 구글에게 아무런 정보를 주지 않습니다.

SEO 파일명 작성 원칙 5가지
  1. 핵심 키워드를 파일명 앞에 배치 — 구글은 앞부분 키워드에 더 높은 가중치를 부여합니다. 예: 대구맛집-파스타-레스토랑.webp
  2. 단어 구분은 하이픈(-)으로 — 언더바(_)보다 하이픈이 구글에서 단어 구분자로 더 잘 인식됩니다.
  3. 한국어 키워드 사용 — 한국어 검색이 주 타깃이라면 한국어 파일명이 한국 구글 검색에서 유리합니다.
  4. 너무 길지 않게 — 5단어 이내를 권장합니다. 과도하게 긴 파일명은 스팸으로 인식될 수 있습니다.
  5. 특수문자·공백 사용 금지 — 공백은 URL에서 %20으로 인코딩되어 지저분해집니다. 하이픈만 사용하세요.
💡 압축짤 파일명 키워드 기능 활용법
블로그 포스팅 주제 키워드를 입력란에 넣으면 모든 이미지에 자동으로 순번이 붙습니다.
예: 대구맛집 입력 → 대구맛집-1.webp, 대구맛집-2.webp, 대구맛집-3.webp …
한 포스팅의 이미지를 한 번에 처리할 수 있어서 시간을 크게 절약할 수 있습니다.
파일명과 함께 챙겨야 할 것 — alt 태그
파일명 최적화와 함께 alt(대체 텍스트)도 반드시 작성하세요. alt 태그는 구글이 이미지를 이해하는 데 파일명보다 더 강력한 신호입니다. 블로그 에디터에서 이미지를 클릭하면 alt 입력란이 나타납니다.

좋은 alt 예시:
  • 대구 동성로 이탈리안 레스토랑 트러플 파스타
  • 홍대 브런치 카페 에그베네딕트 메뉴판
나쁜 alt 예시:
  • 이미지, 사진1, 클릭
