📖 이미지 최적화 가이드 03
티스토리·네이버 블로그 이미지 용량 제한 총정리
이미지 업로드 오류를 겪어보셨나요? 국내 주요 블로그 플랫폼별 이미지 용량 제한과 권장 설정, 해결 방법을 정리했습니다.
2025년 7월
플랫폼별 이미지 제한 총정리
🟠 티스토리
파일당 최대 10MB
WebP, JPG, PNG, GIF 모두 지원. 이미지 1장 최대 10MB까지 업로드 가능합니다. 단 고화질 사진을 그대로 올리면 페이지 속도에 영향을 줍니다. 실사용 권장 용량은 200KB~500KB 이하입니다. 티스토리는 업로드 후 자체적으로 이미지를 리사이즈하지 않으므로 미리 압축해서 올리는 것이 중요합니다.
🟢 네이버 블로그
파일당 최대 10MB
네이버 블로그도 WebP를 지원하며 파일당 10MB 제한이 있습니다. 단, 네이버 블로그는 업로드 시 자체적으로 이미지를 일정 크기로 리사이즈하는 경우가 있어 원본 화질이 변할 수 있습니다. 업로드 전 최적 크기로 줄여두면 리사이즈 없이 원하는 화질 그대로 표시됩니다.
🔵 워드프레스 (WordPress.com 무료)
파일당 최대 1MB총 3GB
WordPress.com 무료 플랜은 파일당 1MB로 제한이 엄격합니다. 스마트폰 사진을 그대로 올리면 대부분 실패합니다. 압축짤로 미리 500KB 이하로 줄여두세요. 자체 호스팅(WordPress.org)은 서버 설정에 따라 다르며 보통 2MB~8MB입니다.
⚫ 브런치 (Brunch)
파일당 최대 5MB
브런치는 JPG와 PNG를 주로 지원하며 WebP 지원 여부는 업로드 시 확인이 필요합니다. 긴 글보다 에세이 형식이 많아 이미지 수가 적은 편이지만, 대표 이미지의 품질이 중요하므로 화질을 유지하면서 용량을 줄이는 게 핵심입니다.
용량 제한에 걸렸을 때 해결 방법
이미지 업로드 오류가 발생하는 이유는 대부분 파일 용량 초과입니다. 아래 순서로 해결하세요.
  1. 압축짤에서 WebP로 변환 — 대부분의 경우 50% 이상 용량이 줄어들어 업로드 제한을 통과합니다.
  2. 출력 포맷을 JPG로 변경 — WebP를 지원하지 않는 플랫폼이라면 JPG를 선택하세요.
  3. 이미지 크기 확인 — 압축짤은 가로 1920px를 초과하면 자동으로 리사이즈합니다. 블로그 본문용 이미지는 1200px 이하로 충분합니다.
  4. GIF 이미지 주의 — GIF 애니메이션은 파일 크기가 매우 커지기 쉽습니다. 짧은 루프 영상이라면 GIF 대신 mp4 동영상으로 대체하는 것을 검토하세요.
✅ 플랫폼별 권장 이미지 크기
• 티스토리: 가로 860px, 200~400KB 이하
• 네이버 블로그: 가로 966px, 200~400KB 이하
• 워드프레스 무료: 가로 1200px, 500KB 이하
• 공통 권장: WebP 포맷, 품질 80~85%
💡 자주 실수하는 것
스마트폰으로 찍은 사진은 기본적으로 4000×3000px, 4~8MB입니다. 이를 그대로 올리면 플랫폼 제한에 걸리거나 페이지 로딩이 극도로 느려집니다. 포스팅 전 항상 압축 과정을 거치는 습관을 들이세요.
